# Agent Rules & Instructions

Har session aur coding tasks ke dauran in rules ko follow karna zaroori hai:

1. **Mobile-First Responsiveness**:
   - Website hamesha mobile-first responsive design ko follow karegi. Coding aur styling pehle mobile screen layouts ke liye likhi jayegi, uske baad media queries ke zariye desktop aur tablet layouts ko enhance kiya jayega.

2. **Hindi & Simple Language for Implementations**:
   - Jo bhi implementation plans ya files (`implementation_plan.md`, `walkthrough.md`, `task.md`, etc.) banegi, wo proper Hindi aur aasan/simple understanding language (Hinglish/Hindi) me likhi jayegi taaki samajhne me aasan ho.

3. **Detailed & Simple Explanations for Code Changes**:
   - Har ek code change ke saath aapko clear and simple words me batana hoga:
     - **Kya**: Kya change kiya gaya hai?
     - **Kyun**: Ye change kyun kiya gaya?
     - **Kisliye**: Is change ka maksad kya hai?
     - **Benefits**: Is change se kya faayde honge?
     Wo bhi simple bhasaa (Hindi/Hinglish) me.

*Note: Future me in rules me aur bhi points add kiye ja sakte hain.*
